김오즈

안녕하세요. 저는 웹 개발자 김오즈. 다양한 웹 기술을 다루며 사용자 친화적인 웹 애플리케이션을 만드는 것을 좋아합니다.

취미

저는 독서를 좋아하며, 주말에는 도서관이나 카페에서 책을 읽는 시간을 즐깁니다. 또한 신선한 재료로 만든 초밥을 좋아하며 새로운 초밥집을 찾아다니는 것을 즐깁니다.

좋아하는 음식

  • 초밥: 신선한 재료로 만든 초밥을 좋아합니다.
  • 파스타: 다양한 소스와 재료로 만든 파스타를 즐깁니다.
  • 김치찌개: 매콤한 김치찌개를 좋아하며, 자주 만들어 먹습니다.

전공

대학에서 컴퓨터 공학을 전공하였으며, 다양한 프로그래밍 언어와 알고리즘을 공부했습니다. 이 과정에서 소프트웨어 개발에 대한 깊은 이해를 쌓았습니다.

프론트엔드 개발을 시작하게 된 이유

웹 페이지가 어떻게 만들어지는지 궁금해서 프론트엔드 개발을 시작하게 되었습니다. 처음에는 HTML이 무엇인지도 잘 몰랐지만, 간단한 웹 페이지를 만들면서 흥미를 느꼈습니다. 사용자들이 웹 페이지를 어떻게 사용하는지 알아가는 과정이 재미있어서 계속 배우고 싶어졌습니다.
