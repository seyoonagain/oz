### 반복문

- for문

  ```js
  for(초기식; 조건식; 반복식) {
      조건이 true인 경우 반복될 로직
  }
  ```

  초기식에서 선언한 변수는 반복문 바깥에선 쓸 수 없다.

- while문

  ```js
  while(조건){
      조건이 true인 경우 반복될 로직
  }
  ```

  조건에 변수가 쓰인 경우, 변수에 대한 변화도 함께 작성해준다.  
   그렇지 않으면 변수가 변하지 않아 계속 같은 조건이기 때문에  
   해당 조건이 true인 경우 무한 반복되게 된다.
